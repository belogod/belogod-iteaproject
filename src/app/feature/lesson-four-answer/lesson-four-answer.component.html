1. Why should we use getter and setter instead of just setting the variable in the service?

2. How to set a default value for @Input() property?

3. What is the difference between @ViewChild() і @ContentChild()?

4. Why do we add providedIn: ‘root’?

1. Геттеры и сеттеры могут разрешать разные уровни доступа - например, get может быть общедоступным, но набор может быть защищен.

2. @Input() property: string = 'default';

3. ViewChild - помогает получить доступ к дочернему компоненту, директиве, элементу DOM внутри родительского компонента.
ContentChildren - это декоратор параметров, который используется для выборки QueryList элементов или директив из DOM содержимого.

4. При добавлении providedIn: 'root' сервис будет доступен для всего приложения как singleton без необходимости добавлять его в массив модуля. Но если он никогда не используется, он не будет включен в сборку
