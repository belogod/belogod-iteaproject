1. What is the difference between structural and attributive directives?

2. What is trackByFn?

3. What is the difference between ngIf and display: none?

4. Explain decimal pipe

1. Структурные директивы используются для формирования или изменения формы HTML DOM путем добавления и удаления элементов.
Директивы атрибутов используются для изменения внешнего вида или поведения элемента DOM.

2. При использовании ngFor для зацикливания массива в шаблонах, если обратиться к функции trackBy, она вернёт уникальный идентификатор для каждого элемента. Angular будет понимать, какой элемент изменился, а затем внесёт изменения в DOM только для этого конкретного элемента.

3. ngIf - позволяет удалить или, наоборот, добавить элемент при определенном условии. При условии === false, элемен полностью удаляется из DOM.
display: none - элемент удаляется из дерева доступности. Это свойство скрывает только визуальную часть элемента, в то время как его можно изменить через консоль.

4. Angular Decimal Pipe - это один из элементов конвейера в Angular, который используется для форматирования десятичных чисел в соответствии с заданной информацией о десятичных цифрах и информацией о языковом стандарте.
