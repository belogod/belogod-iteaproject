1. Explain everything you can see in the package.json and what it is the purpose of the file.

2. What is npm and how do we use that?

3. What is the purpose of the assets folder, main.ts, angular.json, index.html

4. Give an example of smart and dumb components

1. package.json - файл, в котором содержится информация о даном приложении:
- название;
- версия;
- зависимости;
- скрипты.

2. npm - менеджер пакетов, который входит в состав Node.js.

3. index.html - главная страница приложения.
main.ts - скрипт, который выполняется после загрузки страницы.
angular.json - главный конфигурационный файл.

4. Dumb components:
- независим от контекста и бизнес-логики;
- содержит рыбный текст-заглушку, который можно заменить чем угодно;
- иллюстрирует визуальное оформление;
- иллюстрирует возможные состояния и поведение.
Smart components:
- существует в контексте: в определенном экране/блоке/фиче;
- содержит контекст, подходящий для места применения;
- иллюстрирует как работает конкретная фича;
- содержит только необходимые состояния и поведение.
